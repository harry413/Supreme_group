## Features

- Responsive design
- Accessibility compliant
- High performance
- Animations and interactions

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion

## Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Build for production: `npm run build`
